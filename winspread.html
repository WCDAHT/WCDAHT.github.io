WinSpread

Windows Terminal Spreadsheet Calculator

Version 0.8.0 - June 2025

A powerful, vi-style spreadsheet application that runs entirely in the Windows terminal. WinSpread combines traditional spreadsheet functionality with keyboard-driven navigation for efficient data analysis and calculations without leaving the command line.

Documentation

üöÄ Quick Start

Basic navigation and data entry:

  • h/j/k/l - Move left/down/up/right
  • = - Enter formula mode
  • " - Enter text mode
  • : - Enter command mode
  • Ctrl+C/V - Copy/paste cells
  • x - Clear current cell

üìä Data Management

  • :savecsv filename - Save to CSV
  • :loadcsv filename - Load from CSV
  • Ctrl+Shift+1-5 - Apply formatting
  • Alt+Arrows - Resize columns/rows
  • :clrtx color - Set text color
  • :clrbg color - Set background color
