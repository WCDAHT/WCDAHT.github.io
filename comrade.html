COMRADE

Connection Object Mapping and Relational Assessment Database Engine

Version 0.7.2 - July 2025

A modern, interactive Python GUI application for visualizing and managing complex relationship networks between people, textboxes, and legend cards. COMRADE provides an intuitive interface for creating, editing, and exploring connection networks with beautiful, modern design and powerful data management capabilities - perfect for crime analysts, investigators, and relationship mapping professionals.

Installation

üêç Python Application

Run directly from Python source:

  1. Download the source code from releases page
  2. Extract to your preferred directory
  3. Install dependencies: pip install -r requirements.txt
  4. Run: python main.py

üîß Development Setup

Clone and contribute to the project:

# Clone the repository git clone https://github.com/BitEU/COMRADE.git cd COMRADE # Install dependencies pip install -r requirements.txt # Run the application python main.py

üìã System Requirements

  • Python 3.7 or higher
  • tkinter (usually included with Python)
  • Pillow (PIL) for image handling and PNG export (recommended)
  • Windows/Linux/macOS compatibility
  • 2GB RAM minimum (4GB recommended for large networks)
  • 500MB free disk space
